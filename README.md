# spring-boot-starter-application
A starter Spring Boot application designed primarily as a sandbox.

# Technologies
 - Spring (Boot, Data REST, Actuator)
 - Gradle
 - Mongo
 - Lombok
 - JUnit
 - Mockito
 - RestAssured
 
# Setting Up (Linux Debian)

1. Install Java

2. Install Mongo DB

3. Install Spring Tools

4. Install Lombok

5. Clone the project
